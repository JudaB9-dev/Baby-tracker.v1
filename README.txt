גרסה v4 - גרסה מלאה עם תיקון שדה התאריך
=========================================

מה עודכן:
- נשמרו כל הפיצ'רים של הגרסה המלאה בלי אזור התאמה אישית
- תוקן שדה התאריך כדי שלא יגלוש החוצה באייפון
- נוספו התאמות רוחב ומניעת overflow במובייל

איך לעדכן:
1. חלץ את ה-ZIP
2. העלה והחלף את הקבצים הישנים ברפוזיטורי
3. בצע Commit
4. פתח את האתר ורענן
5. אם צריך, הסר והוסף מחדש למסך הבית
