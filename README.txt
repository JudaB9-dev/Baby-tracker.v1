גרסה v5 - גרסה מלאה עם התאמת UI לכל השדות
============================================

מה עודכן:
- נשמרו כל הפיצ'רים של הגרסה המלאה בלי אזור התאמה אישית
- שופרו כל השדות והכפתורים למובייל
- תוקן שדה התאריך
- הוגדרו רוחב, גובה, box-sizing, overflow, ו-grid mobile-friendly
- האפליקציה נשארת אותה אפליקציה, עם אותם פיצ'רים

איך לעדכן:
1. חלץ את ה-ZIP
2. העלה והחלף את הקבצים הישנים ברפוזיטורי
3. בצע Commit
4. פתח את האתר ורענן
5. אם צריך, הסר והוסף מחדש למסך הבית
